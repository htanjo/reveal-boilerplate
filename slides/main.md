# reveal-boilerplate
Boilerplate for [revela.js](http://lab.hakim.se/reveal-js/) presentation.



## Features
- **Quick installation** via [npm](https://www.npmjs.com/)
- **Markdown** support
- **Live preview** using [Browsersync](http://www.browsersync.io/)
- **Easy publishing** to github.io
